# 输入法词条评估详细标准

## 一、词条分类体系

### 基础分类（必填，单选）

| 分类 | 说明 | 示例 |
|------|------|------|
| 名词 | 人物、事物、概念名称 | 手机、电脑、人工智能 |
| 动词 | 动作、行为描述 | 运行、计算、处理 |
| 形容词 | 性质、状态描述 | 快速、高效、稳定 |
| 副词 | 程度、方式、时间修饰 | 非常、已经、突然 |
| 代词 | 指代人、事物、处所 | 我、这个、哪里 |
| 数词 | 数量表达 | 一、百、千万 |
| 量词 | 计量单位 | 个、条、次 |
| 拟声词 | 模拟声音 | 哗、砰、叮咚、汪汪 |
| 叹词 | 表达情感或应答 | 哎呀、嗯、哦 |
| 成语 | 四字成语、典故 | 一举两得、画龙点睛 |
| 专有名词 | 人名、地名、机构名 | 北京大学、腾讯、微信 |
| 短语 | 常用搭配、惯用语 | 打交道、在所难免 |
| 网络用语 | 网络流行语、缩写 | yyds、emo、绝绝子 |
| 其他 | 无法归入以上类别 | - |

### 其他标签（可选，可多选）

| 标签 | 说明 |
|------|------|
| `#口语` | 口语化表达 |
| `#书面语` | 正式书面用语 |
| `#方言` | 方言词汇 |
| `#专业术语` | 特定领域术语 |
| `#新词` | 近年新出现的词汇 |
| `#生僻` | 使用频率较低的词汇 |
| `#多音字` | 存在多音字 |
| `#易错` | 容易写错或用错 |
| `#繁体` | 繁体字词条，建议转换 |
| `#港台用语` | 港台地区特有用语 |
| `#英语` | 英语原文词条（如：June、Amazon、hello） |
| `#日语罗马音` | 日语罗马音词条，用拉丁字母拼写（如：kawaii、sugoi、baka） |
| `#和字` | 日语汉字词汇，包含日本特有汉字或现代汉语不常用字（如：馬鹿、冰菓、牛丼） |
| `#法语` `#德语` `#俄语` `#韩语` 等 | 英语、日语之外的其他语言原文词条，也都要标注对应的语言标签（如：bonjour、danke、спасибо、안녕） |

## 二、评估维度

### 常用性评分标准

| 等级 | 描述 | 示例 |
|------|------|------|
| 高 | 日常频繁使用 | 你好、谢谢、手机 |
| 中 | 特定场景常用 | 算法、协议、服务器 |
| 低 | 极少使用 | 生僻字、古语词 |

### 规范性检查项

1. **字词正确性**
   - 无错别字
   - 符合现代汉语规范
   - 写法统一标准

2. **格式规范性**
   - 无多余空格
   - 无特殊符号
   - 标点使用正确

### 完整性判断

- ✅ 完整词汇：苹果、北京大学、云计算
- ❌ 残缺片段：苹、北京大、云计

### 歧义性评估

需要特别关注的歧义类型：
1. **同音异义**：期中/期终
2. **多义词**：银行（河流/金融机构）
3. **形近字**：己/已/巳

### 专有名词知名度评估

对**人名、地名、机构名、交通设施与地标**等专有名词，需评估其知名程度：

#### 人名知名度

| 等级 | 描述 | 示例 | 建议 |
|------|------|------|------|
| 高 | 广为人知的历史人物、公众人物 | 李白、毛泽东、鲁迅 | 推荐收录 |
| 中 | 特定领域知名人物、知名作品的知名角色 | 莫言、钟南山、马化腾、孙悟空 | 可收录 |
| 低 | 普通人名、不知名人物 | 某普通姓名 | 不建议收录 |

#### 地名知名度

| 等级 | 描述 | 示例 | 建议 |
|------|------|------|------|
| 高 | 省会、著名城市和地区、知名景点 | 北京、上海、中关村、西湖 | 推荐收录 |
| 中 | 一般景点、著名作品和热梗中的知名地点 | 昆山、周庄、大明湖 | 可收录 |
| 低 | 普通乡镇、村庄、不知名地点 | 某村名 | 不建议收录 |

#### 机构名知名度

| 等级 | 描述 | 示例 | 建议 |
|------|------|------|------|
| 高 | 知名企业、高校、政府机构 | 腾讯、北大、教育部 | 推荐收录 |
| 中 | 行业内知名机构 | 某行业龙头企业 | 可收录 |
| 低 | 小型企业、不知名机构 | 某小公司名 | 不建议收录 |

#### 交通设施与地标知名度

评估的核心原则：**高分来自日常书写中的高频使用需求，而非设施的行政级别或工程规模。** 需重点关注词条的**命名独立性**和**日常书写场景出现频率**。

| 等级 | 描述 | 示例 | 建议 |
|------|------|------|------|
| 高 | **有独立辨识度的城市地标性道路/桥梁**，经常出现在地址、定位、游记、新闻报道等日常书写场景中 | 长安街、南京路、外滩、港珠澳大桥 | 推荐收录 |
| 中 | **区域性交通干道或重要枢纽**，在特定城市或场景的书写中有一定需求 | 深南大道、天府大道、北京西站、浦东机场 | 可收录 |
| 低 | **通名主导的泛化交通设施名** 日常逐字输入的需求极低；以及普通街道、小巷、门牌号级道路 | 京港澳高速、XX环城公路、G107国道 | 不建议收录 |

## 三、排除标准

以下类型词条应标记为不适合：

### 1. 错误类
- 明显错别字
- 错误拼写
- 语法错误

### 2. 无效类
- 纯数字组合
- 乱码字符
- 空白内容

### 3. 冗余类
- 重复词条
- 同义词（保留常用形式）
- 过长表述

## 四、特殊词条处理

### 方言词汇
- 标注方言来源
- 评估使用范围
- 注明适用地区

### 网络新词
- 评估生命周期
- 判断接受程度
- 考虑时效性

### 专业术语
- 标注专业领域
- 评估普及程度
- 注明使用场景

## 五、评分标准

### 适用性评分细则（0-100分）

| 分数段 | 评级 | 说明 | 建议 |
|--------|------|------|------|
| 90-100 | 极佳 | 极常用、规范、无歧义 | 强烈推荐收录 |
| 70-89 | 优秀 | 较常用、基本规范 | 推荐收录 |
| 50-69 | 一般 | 有一定使用价值 | 可考虑收录 |
| 30-49 | 较差 | 存在明显问题 | 不建议收录 |
| 0-29 | 很差 | 严重问题或无效词条 | 不推荐收录 |

### 置信度评分细则（0-10分）

| 分数段 | 说明 | 建议 |
|--------|------|------|
| 8-10分 | 判断依据充分，结果可靠 | 直接采纳评估结果 |
| 5-7分 | 判断依据较充分，结果较可靠 | 可采纳，必要时人工复核 |
| 0-4分 | 判断依据不足 | 必须人工确认 |

### 评分参考因素

**高分因素（加分）：**
- 日常高频使用
- 简体中文规范书写
- 常用外语词汇（包括英语、日语假名/罗马音、韩语等广泛使用的外语词）
- 词义明确无歧义
- 广泛认可的词汇

**低分因素（减分）：**
- 含有错别字、拼写错误
- 异体词（非规范字形，如繁体字、异体字形式的词条）
- 残缺不完整、没有构成有效词条
- 容易产生歧义
- 使用场景极为有限（非因内容敏感，而是因词汇本身不常用）

### 内容中立原则

输入法词典服务于用户的书写需求，应保持内容中立。即使词汇涉及粗俗、成人内容等，只要该词汇在实际交流中有使用需求，就应正常评估其收录价值，不应因内容敏感而降低评分或拒绝收录。

## 六、地区与字形规范

### 简体中文标准

评估以中国大陆简体中文为标准：
- 字形：使用简化字规范字形
- 词汇：优先采用大陆通用词汇
- 用法：符合大陆语言使用习惯

### 特殊情况处理

**繁体字词条：**
- 标注 `#繁体` 标签
- 建议提供简体转换形式
- 评分考虑转换后的适用性

**港台用语：**
- 标注 `#港台用语` 标签
- 说明与大陆用语的差异
- 评估在大陆场景的适用性

**方言词汇：**
- 标注 `#方言` 标签
- 说明方言来源地区
- 评估推广使用价值
